# Portafolio Personal

Este repositorio contiene la planeacion y futura implementacion de un portafolio personal para desarrollador construido con `Next.js`.

## Objetivo

Construir un portafolio profesional que en su primera version presente la informacion esencial del desarrollador y sus proyectos, pero que tambien quede preparado para crecer con nuevas secciones como blog, casos de estudio y contenido tecnico.

## Stack previsto

- `Next.js`
- `React`
- `App Router`
- `TypeScript`
- CSS por definir durante la implementacion

## Estado actual

El repositorio ya fue inicializado con `Next.js`, `TypeScript` y `App Router`. Tambien incluye una base visual inicial, contenido local de ejemplo y rutas para el listado y detalle de proyectos.

## Comandos principales

```bash
npm install
npm run dev
npm run lint
npm run build
```
