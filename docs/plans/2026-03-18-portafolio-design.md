# Diseno aprobado del portafolio

## Decision principal

El proyecto se construira con `Next.js` porque permite empezar con una primera version simple y escalar luego sin rehacer la base tecnica.

## Objetivo del producto

Crear un portafolio personal para desarrollador que sirva como presentacion profesional, exhibicion de proyectos y base futura para contenido tecnico.

## Enfoque elegido

Se adopta un enfoque escalable desde el inicio:

- V1 simple
- arquitectura preparada para crecer
- contenido inicialmente local
- SEO y rendimiento considerados desde la base

## Arquitectura acordada

- `Next.js` con `App Router`
- paginas estaticas por defecto
- rutas dinamicas para proyectos en el futuro cercano
- uso de `metadata` y `generateMetadata`
- uso de `next/image` y `next/font`

## Documentacion necesaria

El proyecto debe contar con documentacion de vision, requerimientos, decisiones tecnicas, arquitectura, estructura, modelo de contenido, diseno, SEO, roadmap, desarrollo y plan de implementacion.

## Alcance de V1

- presentacion principal
- sobre mi
- proyectos destacados
- habilidades
- contacto
- responsive
- SEO tecnico base

## Crecimiento previsto

- paginas de detalle por proyecto
- blog o notas tecnicas
- casos de estudio
- ampliacion del contenido profesional
